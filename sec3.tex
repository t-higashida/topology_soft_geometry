\定義{3.7} $K,L$を2次元複体とします．Kの$0-$単体に$L$の$O-$単体を対応させる写像
$ f: K \to L$について，$| xyz|$ が $K$の$2$-単体なら$ | f(x) f(y) f(z) |$が
$L$の単体 ($2$-単体でなくてもかまいません），
また$|xy|$が$K$の$1-$単体なら $ | f(x)f(y) |$ が$L$の単体となるとき，
$f$を単体写像といいます．$ f : K  \to L$ なる単体写像が全単射のとき
$f$を単体同型写像，また$K$と$L$とは同型であるといって
\[
	K \simeq L
\]
と書きます．第2章の定義を思い出してください．これらの定義は第4章で一般の形で考察
することになります．

さて，
\[
	K \simeq L
\]
かつ$K$が閉曲面なら，$L$も閉曲面であることは明らかでしょう．

\問題{3.2}
このことを単体同型写像 $f : K  \to L$を用いてきちんと証明してください．

\証明

$f : K \to L$を単体同型写像とする．$L$の任意の点を$y$とし，
$f(x)  = y$となる$K$の頂点$x$をとる．
$ \mathit{ lk} ( x; K )$は閉多角形，かつ，$f$を$[mathit{lk} (x;K)$に制限したのは，
\[
	\mathit{lk} (x; K ) \simeq f( \mathit{lk} ( x ; K))
\]
なる単体同型を与える．ここで
\[
	\mathit{lk} (y;L ) = f( \mathit{lk} ( x ; K))
\]
を示せば，$\mathit{lk} (y;L)$も閉多角形となる．

\証明

$	\mathit{lk} (y ; L) \ni \Delta$とすれば，$y * \Delta$は$L$の単体であるから，
\[
	f(x * \Delta') = y * \Delta
\]
となる$K$の単体$\Delta'$があり，
\[
	f ( \Delta ') = \Delta
\]
である．
\[
	\therefore \Delta \in f( \mathit{lk} ( x; K)).
\]
逆に$	f( \mathit{lk} ( x; K)) \ni \Delta$とすれば，
\[
	f ( \Delta') = \Delta , \quad \Delta \in \mathit{lk} ( x; K)
\]
となる$ \Delta '$があり，$ x * \Delta'$は$K$の単体である．よって
\[
	f ( x * \Delta') = f(x) * f ( \Delta ' ) = y * \Delta.
\]
は$L$の単体となり，$\Delta \in \mathit{lk} ( y ; L)$である．
